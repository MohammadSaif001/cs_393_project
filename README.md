# Spam Mail Classifier

A compact machine-learning workflow to classify emails as **Spam (1)** or **Ham (0)**.

## Features
- Data cleaning  
- Label encoding  
- TF-IDF feature extraction  
- Logistic Regression model  
- Accuracy evaluation  
- Simple prediction function

## Requirements
```
pip install pandas numpy scikit-learn
```

## How to Use
1. Keep `mail_data.csv` in the same directory.  
2. Run the notebook step‑by‑step.  
3. Use the last cell to test any custom email.

## Files
- `spam_mail.ipynb` — notebook  
- `mail_data.csv` — dataset
